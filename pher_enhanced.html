🌾 Core Concept

The core concept of your system is to use swarm intelligence inspired by insect pheromone behavior to enable smart and efficient pesticide application in farms. In nature, insects like ants leave pheromone trails to guide others toward important locations; similarly, your system creates a digital pheromone map where areas with high pest activity are assigned higher values. Multiple autonomous agents (like drones) use this map to move toward problem zones and spray pesticides only where needed. As spraying happens, the “pheromone” value decreases, preventing repeated or unnecessary spraying. The main goal is to achieve precision agriculture—reducing pesticide waste, minimizing environmental impact, and ensuring crops are treated accurately and efficiently without relying on centralized control.

📟 Arduino Code — ESP8266 (Upload to Board)
// AiVara IoT Module — ESP8266 + Pot + LED + Buzzer // Sends soil moisture to FastAPI backend every 3 seconds #include <ESP8266WiFi.h> #include <ESP8266HTTPClient.h> const char* ssid = "YOUR_WIFI_SSID"; const char* password = "YOUR_WIFI_PASS"; const char* server = "http://<YOUR_PC_IP>:8000/sensor"; #define LED_PIN D1 // GPIO5 #define BUZZER_PIN D2 // GPIO4 #define POT_PIN A0 // Analog input const int ALERT_THRESHOLD = 500; // ADC value (~50% moisture) void setup() { Serial.begin(9600); pinMode(LED_PIN, OUTPUT); pinMode(BUZZER_PIN, OUTPUT); WiFi.begin(ssid, password); while (WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.println("\nWiFi connected!"); } void loop() { int raw = analogRead(POT_PIN); // 0–1023 int pct = map(raw, 0, 1023, 0, 100); // Convert to % Serial.print("Moisture: "); Serial.println(pct); // Local hardware alert if (raw < ALERT_THRESHOLD) { digitalWrite(LED_PIN, HIGH); digitalWrite(BUZZER_PIN, HIGH); // Too dry → alert } else { digitalWrite(LED_PIN, LOW); digitalWrite(BUZZER_PIN, LOW); } // Send to AiVara backend if (WiFi.status() == WL_CONNECTED) { HTTPClient http; WiFiClient client; http.begin(client, server); http.addHeader("Content-Type", "application/json"); String body = "{\\"moisture\\":" + String(pct) + "}"; int code = http.POST(body); Serial.print("HTTP: "); Serial.println(code); http.end(); } delay(3000); }
